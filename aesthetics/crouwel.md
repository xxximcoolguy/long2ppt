# Wim Crouwel · 视觉操作系统

> 「**我必须永远不要站在信息与接收者之间。**」 — Crouwel, 1972 Fodor 辩论
> 「字体不是给人读的，是给行业讨论用的。」 — New Alphabet 自述

| 项 | 内容 |
|---|---|
| **人物** | Wim Crouwel (1928-2019) |
| **身份** | Total Design 联合创始人（1963）/ Stedelijk 首席设计师（1963-85）/ "Mr. Gridnik" |
| **代表作** | Stedelijk 海报系列 · New Alphabet (1967) · Gridnik 字体 (1974) · Fodor 杂志 · Dutch PTT 邮票 |
| **核心立场** | 1972 与 Jan van Toorn 的著名 Fodor 辩论 —— **设计史"系统派 vs 表达派"分野** |

## 1. 视觉哲学

### MM-1 · 设计师是信息的透明导管
【一手 1972 Fodor 辩论】"我必须永远不要站在信息与接收者之间。" 情绪、个性、政治立场都属于"噪音"。与同年代 Brody / Carson 的"设计即表达"**180° 对立**。

### MM-2 · 视觉结构主义
受 De Stijl / Bauhaus 影响。**让网格本身成为视觉主角**，不只是用网格。
> "I imagined a book or publication as a 3 dimensional product, where each position stood in a specific relation to any other position."

### MM-3 · 单位制思维（Unit-Based Design）
Gridnik 字体（1974，为 Olivetti）字干宽度=**7 的倍数**：light=28 / regular=56 / medium=84 / bold=112。
**字号、行高、间距全部用基本单位的整数倍生成，不允许"凭感觉"**。工业设计师对平面的入侵。

### MM-4 · 字体即技术宣言（New Alphabet, 1967）
为早期阴极射线管屏幕设计，**只用横线 + 竖线 + 45° 斜线**，建立在 5×9 网格上。
Crouwel 自承："the typeface was not fit for use, but that it was designed for the sake of discussion."
**设计史上最著名的"概念字体"**。

### MM-5 · 反作者性立场
1972 Fodor 辩论：van Toorn 主张设计师必须带政治立场和主观介入；Crouwel 坚持中性、理性、为客户服务。**奠定"系统派 vs 表达派"两大设计哲学分野**。

## 2. 版式操作系统

| 手法 | 操作要点 |
|---|---|
| **极致网格分割** | 12-16 列网格起步，所有元素严格对齐，**不允许任何元素跨越网格边界**（与 Carson 越界美学完全相反）|
| **数字电网风字体** | 字母由横线 + 竖线 + 45° 斜线构成，模拟点阵屏 |
| **单位制 Unit=8** | 字号梯度走整数倍：7px / 14px / 28px / 56px / 112px，间距同理 |
| **像素化字母** | 用网格本身画字（Stedelijk 海报系列），字与版面共用同一套网格逻辑 |
| **超高对齐密度** | 标题/副标/正文/图说全部走**同一条基线网格**（baseline grid），垂直节奏精确到像素 |
| **大留白 + 角落锚定** | 信息常集中在版面一角（左上 / 左下），剩余 60-70% 留白做呼吸 |
| **几何字体定制** | 重要项目自制字体（Fodor 的等宽打字机字体 + 橙底粉点网格背景）|
| **Stedelijk 海报范式** | 1963-85 间 **统一模板**，所有展览海报共享同一网格 / 字体 / Logo 位置 |

### CSS 落地（Unit=8 系统）
```css
:root {
  --unit: 8px;
  --h1: 96px;   /* 12u */
  --h2: 56px;   /* 7u */
  --h3: 32px;   /* 4u */
  --body: 16px; /* 2u */
  --cap: 12px;  /* 1.5u */
  --line: 1.25; /* 字号 × 1.25 = 8 的倍数 */
}
```

## 3. 颜色与字体

### 配色（黑白基底 + 单色强调）

| 类别 | 偏好 | 占比 |
|---|---|---|
| 主色 | 黑 `#000` / 暖白 `#F5F2EA` 或纯白 | 95% |
| 强调色（**一次只用一个**）| Stedelijk 红 `#E30613` / PTT 橙 `#FF6600` / 工业蓝 `#0050A0` | 5% |
| 禁忌 | 渐变 / 阴影 / 高饱和并列 / 情绪化暖色 | — |

### 字体（优先级排序）

1. **New Alphabet (1967)** —— 仅做封面 / 海报主标题视觉符号，**正文绝不可用**
2. **Gridnik (1974)** —— 单位制几何无衬线，标题首选
3. **Foundry Fodor** —— 等宽打字机感，副标题 / 数据标签
4. **Akzidenz-Grotesk / Helvetica / Univers** —— 正文兜底
5. **中文替代** —— **思源黑体 Medium / 苹方 / Inter**（**拒绝任何衬线、宋体、楷体**）

## 4. 视觉禁忌

1. ❌ **反装饰** —— 不用花纹/纹理/做旧/手绘/印章
2. ❌ **反手工感** —— 不要不规则/温度感/人情味（与吕敬人、东方美学完全相反）
3. ❌ **反情绪化** —— 不用强对比制造冲击 / 不用大字号制造压迫 / 不用倾斜制造动感
4. ❌ **反越界** —— 文字不可越出网格 / 不可重叠 / 不可旋转（与 Carson 解构 180° 对立）
5. ❌ **反衬线** —— 拒绝任何 serif / script / display 装饰字体
6. ❌ **反居中** —— 几乎全部左对齐，居中 = "无网格懒人方案"
7. ❌ **反"作者性表达"** —— 设计师个人趣味不能盖过信息本身
8. ❌ **反 3D / 渐变 / 阴影 / 圆角** —— 所有效果必须能用尺规作图复现
9. ❌ **反混排** —— 一个版面最多 2 种字体（通常只用 1 种字体的不同字重）
10. ❌ **反"feel 对就行"** —— 任何决策必须能用网格 / 单位制解释

## 5. 局限

| 场景 | 为什么不适合 |
|---|---|
| **大众消费 / 电商 / 直播** | 过于克制专业，缺乏情绪钩子，转化率差 |
| **情绪驱动**（个人故事 / 鸡汤 / 婚礼）| 中性立场无法承载温度 |
| **儿童 / 教育低龄** | 几何冷感，缺乏亲和力 |
| **传统文化 / 国风 / 文人调** | 与东方书籍美学（杉浦 / 吕）哲学冲突 |
| **正文阅读**（特别 New Alphabet）| 几乎不可读，Crouwel 本人都承认 |
| **中文密度问题** | 中文笔画密度高，纯几何网格易拥挤僵硬，需降级为 "Crouwel-inspired" |
| **营销爆款 / 社媒** | 没视觉冲击力，划走率高 |
| **需要"作者性"** | 中性立场抹平个性 |

**中文降级：** 保留"严格网格 + 单位制 + 黑白 + 单一强调色"，字体降级为思源黑体；New Alphabet 这类不可读字体只做装饰符号。

## 蒸馏到 PPT skill 的核心准则

1. **12-16 列网格起步**，所有元素 snap
2. **Unit=8 系统**：字号 / 间距 / 行高都是 8 的倍数
3. **配色 ≤ 3**：黑 + 白 + 1 个 Stedelijk 红 / PTT 橙 / 工业蓝
4. **字体白名单**：Gridnik / Foundry Fodor / Helvetica / 思源黑 Medium
5. **左对齐为主**，禁止居中（除非纪念碑式标题）
6. **0 装饰 / 0 阴影 / 0 渐变 / 0 圆角 / 0 emoji**
7. **基线网格对齐到 8px**
8. **适用边界**：学术 / 美术馆 / 工业 / 政府；**不用于**营销 / 情绪 / 国风
