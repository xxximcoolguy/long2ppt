# Jan Tschichold · 视觉操作系统

> 「**Das Wesen der neuen Typographie ist Klarheit.**（新排版的本质是清晰）」— *Die neue Typographie*, 1928
> 「新排版的教条"在精神上接近了它本想反对的法西斯主义"」— Tschichold 1959 自我反思

| 项 | 内容 |
|---|---|
| **人物** | Jan Tschichold (1902-1974) |
| **核心冲突** | 前半生新排版革命旗手 · 后半生古典回归 —— **同等严苛标准走完两条相反的路** |
| **代表作** | *Die neue Typographie* (1928) · *Asymmetric Typography* (1935) · Penguin Composition Rules (1947-49) · **Sabon 字体** (1967) |
| **辩证内核** | 不变：严格网格 + 精确字号阶梯 + 偏执细节 · 变：现代/古典的外壳 |

## 1. 视觉哲学

### MM-1 · 清晰是排版的本质
【一手 · *Die neue Typographie* 1928 开篇】
"Das Wesen der neuen Typographie ist Klarheit." 排版是为读者服务的功能性艺术，不是设计师的自我表达。

### MM-2 · 前后期分水岭（1933 流亡瑞士后）
**前期（1923-1933）** —— 不对称是现代精神的视觉表达，对称是 19 世纪的"虚假秩序"。Bauhaus / 构成主义 / El Lissitzky / Moholy-Nagy 影响。

**后期（1946 后）** —— 1959 *Glaube und Wirklichkeit* 自我批判：
> 新排版的教条"在精神上接近了它本想反对的法西斯主义"

承认对称在书籍排版中"千年验证"的合理性；强调**可读性高于风格** —— 长文必须衬线、对称版心、传统比例。

### MM-3 · 辩证统一处（结构理性不变）
前后期表面相反，内核未变：**严格的网格 + 精确的字号阶梯 + 对细节的偏执**。这是 Tschichold 真正的"思维 DNA"。

### MM-4 · Penguin 时期实用主义
《Penguin Composition Rules》(1947-49) 是哲学落地的典范 —— 不追求风格，只追求"500 种书共用一套标准仍能各自妥帖"的**系统性**。

## 2. 版式操作系统

### 双模式（生成时按场景选）

| 场景 | 用 |
|------|---|
| 海报 / 封面 / 章节扉 / 宣言 / 视觉冲击 | **early** 模式（不对称 + Futura/sans + 红黑）|
| 长文 / 书籍 / 学术 / 报告 / 演讲内文 | **late** 模式（对称 + Sabon/serif + 米黑）|

### Early 模式（1923-33 · 新排版）
- **不对称构图**：版心偏左 / 偏上，留白作为构图元素
- **网格强约束**：所有元素吸附于隐形栅格
- **字号严格阶梯**：3-4 级，比例 1:1.5:2.25（黄金比 / 斐波那契近似）
- **sans-serif 革命**：Futura / Akzidenz-Grotesk 取代 Fraktur
- **粗横线分隔**：1pt-6pt 实线水平 / 垂直切割版面
- **DIN 标准纸张**：A4 系列比例

### Late 模式（1947 后 · Penguin / 古典）

**Tschichold Canon（一手版心构造）：**
- 纸张比例 2:3 时版心比例同为 2:3
- **订口:天头:切口:地脚 = 2:3:4:6**
- 天头小、地脚大、订口窄、切口宽（"对称视野中的不对称"）

**Penguin Composition Rules 核心条款：**
- 段落首行缩进 1 em，**章节首段、星号后首段不缩进**
- 标点后**单空格**（反对英美双空格）
- 引号、破折号、省略号精确用法
- 大写字母之间加字距，**小写绝不加**
- 数字与单位之间用窄空格（thin space）

**斐波那契字号比例（后期一手）：**
6/7/8/9/10/11/12/14/16/18/21/24 —— 近似 1.125-1.25 几何增长。

## 3. 颜色与字体

### Early 配色字体
- **配色**：红 + 黑 + 纸白三色。**红占比 < 10%**（标题、横线、点缀）
- **字体**：Futura（Paul Renner, 1927，他亲自校订字距） / Akzidenz-Grotesk / Bodoni（对比时）
- **拒绝**：所有 Fraktur、装饰体、手写体

### Late 配色字体
- **配色**：**米白（cream / off-white）+ 墨黑** + 极少酒红或深蓝
- **反对纯白纸** —— 纯白伤眼，书籍纸张必须微黄
- **字体**：
  - **Sabon (1967)** —— Tschichold 设计的衬线字体。基于 Garamond，比例更稳重，**同字体在 Linotype/Monotype/手工铸字三系统视觉完全一致**
  - Garamond / Bembo / Baskerville（Penguin 时期推崇）
  - Janson（1954 他亲自重制）

### 中文场景映射
- Sabon → 思源宋体 / 方正书宋
- Futura → 思源黑体 Medium
- 字号比例不动，但中文方块字字距设 0.02-0.04em

## 4. 视觉禁忌

### Early 禁忌（1928 一手）
- ❌ **反 Fraktur** —— "民族主义的视觉枷锁"（这条让他被纳粹定罪）
- ❌ **反对称** —— "Symmetrie ist der Ausdruck einer toten Ordnung"（对称是死秩序的表达）
- ❌ **反装饰线框** —— 花边、纹饰、阴影、立体一律拒绝
- ❌ **反居中标题** —— "19 世纪官僚审美"

### Late 禁忌（1946 后一手反转）
- ❌ **反"早期的自己"** —— 公开称早期教条"幼稚且危险"
- ❌ **反纯白纸** —— 长文必须米白
- ❌ **反长文 sans-serif** —— sans 只适合标题/表格/标牌，**绝不书籍正文**
- ❌ **反双空格** —— 句末单空格
- ❌ **反小写加字距**
- ❌ **反未对齐的标点** —— 必须光学悬挂（hanging punctuation）

### 跨时期共同禁忌
- ❌ chartjunk / drop shadow / 渐变 / "为风格而风格"
- ❌ 不基于网格的随意排布

## 5. 局限

1. **介质局限** —— 全部方法论建立在**印刷纸张**上。米白纸、Sabon 金属字模优化、悬挂标点、订口比例 → 屏幕上必须重新映射。直接照搬 PPT 会"形似神不至"
2. **长文偏好** —— 精华在 500 页以上书籍。PPT 是"短文+视觉"，对"一屏 3-5 个要素"几乎无直接论述
3. **西文本位** —— 字距/字号/悬挂标点全基于拉丁字母。**中文方块字直接套用 Sabon 比例 + 中文宋体会失衡**
4. **风格克制** —— 不适合营销爆款、社媒、情绪驱动。需要冲击力调 Carson / Bass
5. **必须选边** —— skill 落地时不能既要早期激进又要后期古典。按内容明确选择

## 蒸馏到 PPT skill 的核心准则

**双模式触发：**
- `tschichold-early` —— 海报 / 封面 / 章节扉：不对称 + Futura/思源黑 + 红黑 + 粗横线
- `tschichold-late`（**默认**）—— 长文 / 内容页：对称版心 + Sabon/思源宋 + 米黑 + 斐波那契字号

**两模式共有铁律：**
1. 字号比例严格 1:1.125:1.25:1.5:1.618:2
2. 网格强约束（隐形或可见）
3. 反 chartjunk / 反装饰 / 反"为风格而风格"
4. 中文场景映射：Sabon→思源宋，Futura→思源黑
5. 段落首字 1em 缩进（章节首段不缩进）
6. 标点单空格 / 光学悬挂
