# 杉浦康平 × 吕敬人 · 东方书籍设计美学操作系统

> 「东方书籍设计 = **在信息丰盈中维持秩序，在严格网格上故意越界，在现代媒介里保存文化身份。**」

| 项 | 杉浦康平 | 吕敬人 |

## 1. 视觉哲学

### 杉浦的核心命题
- **曼荼罗思想**：版面 = 宇宙的微缩。中心—边缘、对称—破对称、显—隐四象限同时呈现
- **杂学（ざつがく）**：拒绝学科切割，天文、神话、声音、字体可在一页共存。**信息越混杂，秩序越显高级**
- **Noise Design 噪音美学**：故意保留"信息毛刺" —— 脚注、批注、异体字、肌理、错位印章。Noise 不是脏，是密度的证据

### 吕敬人的核心命题
- **书籍五感**：视、触、听、嗅、味。投射到 PPT —— 追求"翻页的仪式感"而非"幻灯片的扁平感"
- **书筑学**：以建筑思维做书。封面=门厅，环衬=过道，内文=厅堂。每一页是空间序列中的一站
- **编辑设计观**：设计师是"视觉编剧"，版式是叙事节奏，不是排版工

### 东西方核心区别

| 维度 | 西方现代主义（Swiss）| 东方书籍设计（杉浦/吕）|
|------|---------------------|--------------------|
| 信息密度 | 减法、留白、中性 | **加法、层叠、丰盈** |
| 网格 | 数学纯粹、对称 | 网格 + 故意破网格 |
| 字体观 | 字体 = 容器 | **字体 = 文化身份** |
| 阅读路径 | 线性、Z 字形 | 多入口、跳跃、回环 |
| 留白 | 功能性呼吸 | **"虚"——气韵流动的实体** |

> **关键洞察：东方书籍设计不是"留白少"，而是"留白被理解为另一种实体"（气、虚、白）。**

## 2. 版式操作系统

### H-1 · 高密度多层信息架构
经典杉浦版式 4 层共存：
- **主文层**（正中，最大字号）
- **注释层**（小字号，紧贴或外切主文）
- **边栏层**（窄长条，竖排，与主文垂直）
- **图像/印章锚点层**（朱砂方印、小插图、符号）

```css
.page { display: grid; grid-template-columns: 1fr 280px; gap: 32px; }
.main { font-size: 18px; line-height: 1.9; }
.sidebar { writing-mode: vertical-rl; font-size: 13px;
           border-left: 1px solid #8b6f47; padding-left: 12px; }
.annotation { font-size: 12px; color: #5a4a3a; margin-top: 4px; }
```

### H-2 · 直排 × 横排混合
- 主文横排（现代易读）
- **边栏 / 引文 / 标题用直排**（`writing-mode: vertical-rl`）
- 标点处理：直排时句号、逗号居右上；用 `text-orientation: upright` 让数字与西文回正

### H-3 · 朱砂方印作锚点（最高 ROI 技巧）
朱砂方印是东方版式最强的"视觉标点"。

**使用规则：**
- 颜色：**朱砂红 #B22222 / #A52A2A**，不用纯红 #FF0000
- 位置：标题右下、章节起首、引文末尾
- **一页最多 2 枚**，多则廉价
- 形状：方形 > 圆形 > 椭圆。阴文（白字红底）庄重，阳文（红字白底）轻盈

```css
.seal {
  display: inline-block; width: 64px; height: 64px;
  background: #A52A2A; color: #fff5e6;
  font-family: "SimSun", "STKaiti", serif;
  font-size: 22px; line-height: 1.1; text-align: center;
  padding: 8px; transform: rotate(-2deg);
  box-shadow: inset 0 0 8px rgba(0,0,0,0.3);
  filter: url(#roughen);  /* SVG turbulence 做旧 */
}
```
**做旧的灵魂在于边缘不齐 + 内部斑驳**，可用 SVG `<feTurbulence>` + `<feDisplacementMap>`。

### H-4 · 网格 + 不对称
**先严格定义 12 栏网格，然后故意让 1-2 个元素越界 5-10%**。不是无序，是"有依据的越界"。

PPT 落地：主标题靠左上 1/3 处，**右下配一枚印章压角**，形成对角线张力（不居中）。

### H-5 · 古籍装帧元素现代化
| 古籍元素 | 现代化提示 |
|---------|----------|
| 包背装 / 线装 | 页面左侧加 4 个小圆点（装订孔），距边 24px |
| 经折装 | 横向滚动布局，每"折"=一屏 |
| 版心 / 鱼尾 / 象鼻 | 页眉中央放对称小图标（鱼尾纹）做章节分隔 |
| **天头地脚** | **上下留白远大于左右**（天头 > 地脚 > 左右），打破西方等距留白 |

### H-6 · 宣纸纹理 CSS 模拟
```css
.paper {
  background: #f5ecd9;  /* 宣黄底 */
  background-image:
    radial-gradient(circle at 20% 30%, rgba(139,111,71,0.04) 0%, transparent 40%),
    radial-gradient(circle at 70% 60%, rgba(139,111,71,0.03) 0%, transparent 35%);
}
.paper::before {
  content: ""; position: absolute; inset: 0;
  background-image: url("data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg'><filter id='n'><feTurbulence baseFrequency='0.9' numOctaves='2'/></filter><rect width='100%' height='100%' filter='url(%23n)' opacity='0.06'/></svg>");
  pointer-events: none;
}
```
**关键：底色不用纯白**（#fff 是 PPT 廉价感的元凶），用 `#f5ecd9` / `#f0e6d2` / `#ebe0c8`。

## 3. 颜色与字体启发式

### 东方五色配色
| 名 | HEX | 用法 |
|----|-----|------|
| 朱砂红 | `#A52A2A` / `#B22222` | 印章、强调标题 |
| 墨黑 | `#1a1a1a`（**不用 #000**） | 正文 |
| 宣黄 | `#f5ecd9` / `#ebe0c8` | 底色 |
| 青绿（石青/石绿）| `#2c5f5d` / `#4a7c59` | 副标题、图表 |
| 靛蓝 | `#2c3e50` / `#1e3a5f` | 引文、链接 |
| 赭石 | `#8b6f47` | 边框、分隔线 |

**配色法则：** 一页最多 3 色（墨 + 朱 + 第三色）。第三色按气质选 —— 青绿（雅）、靛蓝（学）、赭石（古）。

### 字体角色分工（中文）
| 字体 | 角色 | 推荐免费版 |
|------|------|----------|
| **宋体** | 正文、理性、信息层 | 思源宋体 / 霞鹜文楷-宋 |
| **楷体** | 引文、注释、人文层 | 霞鹜文楷 / 站酷小薇 |
| **隶书** | 副标题、章节名（少用） | 方正隶变 |
| **篆书** | 印章、装饰（极少 1-2 字） | 方正小篆 |
| **黑体** | 极少用，仅当现代对比 | 思源黑体（仅限标签）|

**禁忌：** 通篇黑体 = 立刻变 Helvetica 化，文化身份归零。

### 中西文搭配
- 中文宋体 + 西文 **Garamond / Bodoni / EB Garamond**（古典衬线）
- 中文楷体 + 西文 **Cormorant Garamond**（更瘦更雅）
- **绝对禁忌：** 中文宋体 + 西文 Helvetica/Arial

## 4. 视觉禁忌

1. ❌ **反 Swiss 中性** —— 拒绝 Helvetica + 12 栏纯净网格 + 大留白。在东方语境下显得"虚无"
2. ❌ **反"现代化即去文化"** —— 去掉印章、直排、宋体的"国际范"中文 PPT = 穿西装拍全家福
3. ❌ **反"信息减法"**（杉浦核心立场） —— TED 风每页一句话+一张图在东方版式观里是浪费
4. ❌ **反"古风装饰" / 假国风**（吕敬人核心立场）—— 水墨笔刷 PNG 贴角 + 祥云边框 + 故宫红配色 = 旅游纪念品。**真国风是结构性的，不是贴图式的**
5. ❌ **反 "Helvetica 化"** —— 吕敬人多次批评当代中国设计照搬 Helvetica + 大留白，称之为"文化失语"
6. ❌ **印章滥用** —— 一页 3 枚以上 = 廉价，等同于到处贴 logo

## 5. 局限性

| 场景 | 是否适用 | 降级方案 |
|------|---------|---------|
| **PPT 演讲投影** | 部分适用 | **必须降级：** 保留印章+宣纸底+宋体，但每页元素降到 5-8 个 |
| **手机小屏** | 不适用 | 直排放弃，多层信息折叠为 tab |
| **快速浏览 / 信息流** | 不适用 | 杉浦哲学要求"慢读"，与刷屏冲突 |
| **西方读者面向** | 谨慎使用 | 印章可作为异文化锚点，但直排+多层需简化 |
| **数据可视化** | 不适用 | Noise 美学干扰数据读取，回归 Tufte |
| **非中文 / 非日文** | 几乎不适用 | 印章、直排、宋体的文化重量无法迁移到拉丁字符 |

**PPT 降级关键 4 元素**：保留 **底色（宣黄）+ 字体（宋楷）+ 印章（1 枚/页）+ 不对称构图**，放弃多层信息+高密度。

## 蒸馏到 PPT skill 的 7 条可执行指令

1. **底色禁纯白**，用 `#f5ecd9` + SVG 噪点
2. **正文用宋体或楷体**，禁黑体通篇
3. **每页 1 枚朱砂方印**（`#A52A2A`），右下或标题角，做旧斑驳
4. **配色 3 色封顶**：墨 + 朱 + 一个第三色（青/靛/赭）
5. **天头大于地脚**，左右等距小于上下（打破等距留白）
6. **标题或引文用直排**（`writing-mode: vertical-rl`），主文横排
7. **网格构图后让一个元素越界 5-10%**，制造"有依据的不对称"
