# Stefan Sagmeister · 视觉操作系统

> 「**Design that creates emotion.** 设计如果不能引起情绪反应，就是失败的。」

## 1. 视觉哲学

### MM-1 · Design that creates emotion
设计如果不能引起情绪反应，就是失败的。**反纯 minimalism / 反 corporate** —— 立场鲜明反对 Dieter Rams / Jony Ive 的"无情感纯净主义"。

### MM-2 · Style = Fart（1999 工作室门口的牌子）
"风格"是设计师懒惰的副产品。**重要的是为每个项目找到独有的视觉答案**，绝不预先选风格。

### MM-3 · 手工感优先（Handmade as proof of presence）
"Typography is proof of human presence" —— 文字是人留在世界上的痕迹，必须可见地"被人做出来"，而不是从字库里调出来。

### MM-4 · Pain = Authenticity
"我们本可以 Photoshop AIGA 海报，但在皮肤上刻字结果更真实，过程更有趣（也更痛）。" 痛苦/不便/麻烦是 Sagmeister 主动选择的方法论 —— **它逼出真实**。

### MM-5 · Sabbatical as method
每 7 年关闭工作室 1 年（1999、2008、2015）。"未来七年所有设计都来自那一年"。**慢即是快**。

### 传承：维也纳学派 → Tibor Kalman / M&Co
Kalman 教给他：① 拒绝抛光的空洞 ② 设计必须带社会观察 ③ 自己开公司才能保住艺术自由

## 2. 版式手法

| 手法 | 操作 | 经典案例 |
|---|---|---|
| **身体作字** | 字直接刻/写/投影在皮肤上 | AIGA Detroit 1999 海报（X-ACTO 刀刻裸躯，Tom Schierlitz 摄影，MoMA / Cooper Hewitt 永久馆藏）|
| **物质媒介作字** | 咖啡渍、香蕉、雪、衣架、热狗、电线、植物、撕碎报纸、灰尘摆字 | "Self Confidence" — 10500 根香蕉摆整面广告牌（绿/黄香蕉色差形成字）|
| **字与图融合** | 文字不是覆盖图，而是**就是图本身** | Lou Reed《Set the Twilight Reeling》文字写满 Reed 脸 |
| **大字 + 手写** | 主标题极大手写体撑满版面，注释缩到很小 | Things I Have Learned 全系列 |
| **拼贴 / 反拼贴** | 大量摄影+手写+现成物，保留"接缝可见"粗糙感 | Talking Heads《Once in a Lifetime》盒装 |
| **不对称 / 不完美** | 故意保留笔迹抖动、材料阴影、操作痕迹 | 全作品 |
| **反规模化** | 每项目独有视觉，绝不复用模板（Things 一书有 15 种封面）|  |

### 对 PPT 的翻译
- 主标题 = 手写大字（Caveat / Permanent Marker / Reenie Beanie / Kalam webfont）撑满 40-60% 版面
- 背景 = 一张高饱和摄影或材质照片，文字**直接压在图上**
- 注释 = Times Roman 反讽地用很小字号
- 接受不对齐 / 笔迹抖动 / 字压住图的脸

## 3. 颜色与字体

### 配色（基于作品归纳）
```css
--rose: #E91E63;        /* 玫红 */
--orange: #FF6B35;      /* 暖橙 */
--banana: #FFD23F;      /* 香蕉黄 */
--blood: #D62828;       /* 血红 */
--skin: #FFC8B4;        /* 粉肤 */
--cobalt: #1E40AF;      /* 钴蓝 */
--grass: #4CAF50;       /* 草绿 */
--paper: #FAF7F2;       /* 纸白 */
--ink: #1A1A1A;
```

**规则：** 单页 2-3 色，至少 **1 个高饱和 + 1 个粉/肤色作软垫**。**拒绝 Pantone 19 系灰、企业蓝、莫兰迪**。

### 字体
- **手写体（主力）**：Caveat / Permanent Marker / Reenie Beanie / Patrick Hand / Shadows Into Light
- **Times Roman / Times New Roman**（**反讽用法**）—— 严肃叙事段落故意用最学院化衬线，与手写形成黑色幽默对照
- **Replica / Replica Mono**（Lineto）—— 工作室长期用
- **中文场景**：霞鹜文楷 / 字魂手写体 / 寒蝉手书体（手写感强）+ 思源宋（反讽 Times Roman）

## 4. 视觉禁忌

1. ❌ **反极简 minimalism** —— Sagmeister 立场鲜明反对 Rams / Jony Ive 纯净主义
2. ❌ **反企业模板** —— "Design with no emotion is design that doesn't matter"
3. ❌ **反纯矢量净化** —— 不要把手写描成 SVG 干净路径，手写的价值在抖动、压力变化、墨水堆积
4. ❌ **反 drop shadow / glassmorphism / neumorphism** —— "为高级而高级"的风格病
5. ❌ **反居中保守** —— 接受偏移 / 悬挂 / 压边 / 出血
6. ❌ **反 stock photo** —— 必须专门拍摄的、有材质、有人在场；不能微笑商务握手
7. ❌ **反统一系统** —— 前后页风格剧变是好的，让节奏推动情绪
8. ❌ **Style = Fart** —— 不要先选风格再填内容；**先有内容再发明视觉**

---

## 5. 局限

1. **企业严肃场合不适用** —— 上市路演、董事会、银行/律所/医疗/政务
2. **数据可视化失效** —— 切 Tufte / Vignelli
3. **大规模可复用模板失败** —— 美学核心是"每页独有"，无法生成 50 页风格统一的产品手册
4. **中文场景挑战** —— ① 没有"手写大字撑满版面"成熟 webfont 生态 ② 中文笔画密度本就高，叠摄影/材质背景易糊 ③ "字与物融合"在中文几乎不可能用真实物体摆出整段
   - **折中：** 中文用大字号开源手写体（霞鹜文楷 / 字魂手写）做标题，正文回正常字号
5. **观众疲劳风险** —— 5-10 页极有效，超过 30 页观众麻木；适合 TED / Pecha Kucha 短演讲
6. **审美门槛** —— 演讲者本人必须有故事 / 有观点 / 有勇气暴露脆弱；标准汇报内容再 Sagmeister 化就是"穿龙袍的假皇帝"

## 蒸馏到 PPT skill 的核心准则

1. **手写体主标题撑满 40-60% 版面**（中文用霞鹜文楷大字）
2. **背景一张高饱和摄影或材质照**，文字直接压在图上
3. **配色 2-3 色，高饱和 + 粉肤色作软垫**
4. **接受不对齐、笔迹抖动、压脸**
5. **每页独有版式**，不复用模板
6. **Times Roman 反讽用法**作为严肃叙事段落字体
7. **适用边界**：TED / Pecha Kucha / 创意 / 文化 / 个人叙事；**绝不**用于企业严肃 / 数据 / 50 页+ 模板
