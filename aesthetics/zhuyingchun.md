# 朱赢椿 · 视觉操作系统

> 「**慢 —— 不是懈怠，是把节奏放慢，让日常与内心不断碰撞。**」 — 朱赢椿工作室门口的牌子
> 「**我希望设计是不动声色的，就像从书里自然长出来的一样。**」

| 项 | 内容 |
|---|---|
| **人物** | 朱赢椿 (b. 1970)，南京师范大学书文化研究中心主任，"书衣坊"工作室 |
| **代表作** | 《不裁》(2007 毛边书)· 《设计诗》(2012)· **《虫子书》(2014, 蜗牛蚂蚁作字, 无一汉字)** · 《肥肉》· 《蚁呓》· 《云朵》 |
| **辈分** | 中国书籍设计第二代代表（吕敬人之后），**更具实验性和个人化** |

## 1. 视觉哲学

### MM-1 · "慢" 作为根本姿态
工作室门口立着他亲手做的交通牌"慢"。这不是装饰，是设计的总开关。
> "慢，不是懈怠、懒惰、消极，而是把节奏放慢，让日常生活与内心不断地碰撞。"

翻译到视觉：**留白不是稀缺，是奢侈；密度不是力量，是噪音；装饰不是丰富，是焦虑**。

### MM-2 · "不动声色" —— 设计的隐身术
> "我希望设计是不动声色的，就像从书里自然长出来的一样。"
> "追求个人风格反而要淡化个人风格 ——'无风格'成为最显著的风格。"

**与吕敬人最大的分野：**
- **吕敬人 = 书筑学**：书是建筑，要有结构、信息层级、阅读路径
- **朱赢椿 = 书生长学**：书是生物，是从纸里长出来的物件，设计师是助产士不是建筑师

### MM-3 · 把书做成"物件"而非"载体"
- 《不裁》(2007)：故意做成毛边书，附纸质裁纸刀，**读者必须一边读一边裁**。书的"完成"由读者参与
- 《虫子书》(2014)：全书无一汉字。在院子里放纸板和深色蔬菜汁"墨池"，蜗牛蚯蚓爬过留下的痕迹即是"字"。作者署名是虫子
- 《设计诗》(2012)：诗的内容用版式本身说出来，**文字就是图像**

哲学：**书不是用来读完的容器，是用来与之相处的物件**。

### MM-4 · 反 corporate 美学
> "现在我想克制，不想再做让人骇人听闻的创意，我想注重感受，想往寡淡无味上走。"

明确反对：流水线美学、KPI 美学、视觉冲击美学、"高级感"模板美学。**"寡淡无味" = 禅宗审美的现代再生**。

## 2. 版式操作系统

### H-1 · 极致留白 60-80%
《蚁呓》典型一页：仅一只小蚂蚁在版面中央偏下，**周围全部空白。蚂蚁旁边一行小字：「我搬不动这粒米」**。
- 留白不是"剩下的地方"，是**主角**
- 文字是闯入留白的客人

**PPT 操作：** 单页信息密度刻意降到他人的 **20%**。一页一句话，甚至一页一个字，是常态而非例外。

### H-2 · 拙朴排版 —— 拒绝"对齐的暴政"
- 字与字之间留呼吸；行与行之间留沉默
- **故意不对齐**：标题往左偏，正文往下沉，模拟手稿的随意
- **故意印不清**：《肥肉》部分页面油墨故意做深浅不匀，模拟旧书残页

### H-3 · 残缺美学
- 《不裁》的毛边：读者裁开后每本书的毛边都不同，**每本都是孤本**
- 部分书页故意做折痕、墨点、虫蛀痕迹（人工模拟自然侵蚀）
> "有一点瑕疵反而更有亲和力。"

### H-4 · 手写 / 涂鸦 / 自然痕迹介入
- 《虫子书》整本由虫子"写"
- 《设计诗》大量手写体、涂改痕迹、批注小字
- **永远不要把版面做到"完美"** —— 完美 = 死亡

### H-5 · 字号极小 + 大留白反差
- 一行 **5pt** 小字配整张空白页 —— 读者必须**凑近**
- 物理层面**强制慢下来**：不能扫读，必须凑近、必须聚焦、必须等待

## 3. 颜色与字体

### 配色（克制到近乎单色）

| 颜色 | HEX | 用途 | 频率 |
|---|---|---|---|
| 宣纸米黄 / 本白 | `#F5F0E6` | 底色主色 | 90% |
| 墨黑 | `#1A1A18` | 文字 / 主图 | 高 |
| 中性灰 | `#A8A6A1` | 辅助文字 / 虫迹 | 中 |
| 朱砂红 | `#B8332A` | 印章 / 极少强调 | **≤ 3%** |
| 草青、土黄 | — | 偶尔点缀 | < 1% |

**禁忌：** 高饱和 / 渐变 / 霓虹 / 品牌色系。

### 字体：宋体为主，几无装饰

- **正文绝对主力**：宋体（中文）/ 衬线体（英文）
- **手写体**：用于诗、批注、虫迹的"翻译"，但极克制
- **黑体禁用区** —— 朱赢椿几乎从不在正文用粗黑体，认为黑体"太用力"
- **字号梯度**：通常**只有 2-3 级，且差距夸张**（如 8pt vs 36pt，中间无过渡）

## 4. 视觉禁忌（红线）

1. ❌ **反"精致"设计** —— 反对修图修到毛孔消失、对齐到亚像素的"高级感"。**瑕疵是诚意**
2. ❌ **反满版** —— 任何页面填满 > 50% 视觉元素 = 背离哲学
3. ❌ **反热闹** —— 多图拼贴 / 信息图密集 / emoji / icon 装饰墙 全禁
4. ❌ **反明亮高饱和** —— 任何 saturation > 60% 颜色都要警惕
5. ❌ **反"视觉冲击力"** —— 朱赢椿设计的书第一眼可能"什么都没有"，**这是故意的**
6. ❌ **反模板化** —— 每本书版式都从内容生长，不复用上一本
7. ❌ **反炫技字体** —— 装饰字体 / 3D 字 / 镂空字 / 霓虹字 全 ban
8. ❌ **反居中对齐的"安全感"** —— 他偏好偏移、错位、下沉
9. ❌ **反 PPT bullet 文化** —— 从不把信息组织成项目符号清单
10. ❌ **反"信息密度即专业"** —— **与杉浦/吕敬人正相反**，应用时需明确选择

## 5. 局限

### 5.1 节奏局限：极慢，不适合演讲投影
- 朱赢椿的设计为**独自阅读、长时间相处**而生
- 观众平均 15-30 秒看一页，他的设计需要 **3 分钟以上**才能"读懂"留白
- 大留白 + 极小字在投影上 = 后排看不见

**解决方案：** 用于深度阅读型电子文档 / Keynote 自习版 / 视频号长文卡片，**不适合 conf talk**。

### 5.2 信息密度局限
- 财报 / API 文档 / 操作步骤需要密集信息，朱赢椿风格会让信息丢失
- **退化方案：** 仅在封面 / 章节过渡 / 引用 / 结语页用朱赢椿风格；正文章节切回 Rams / Tufte

### 5.3 高度个人化，不可批量复用
- 本质是**手作**：手写、手裁、手贴、与虫子合作
- AI 模仿只能学到 70%，剩下 30% 是个人生命经验，无法 prompt 化
- **诚实承认：这套风格 skill 化后只是"调性指引"，不是"风格克隆"**

### 5.4 不适合营销、电商、社媒爆款
- 朱赢椿风格 = **反流量**
- 小红书爆款卡片 / 抖音封面 / 电商详情页 ban

### 5.5 文化错位风险
- "寡淡"建立在中文语境的禅宗 / 文人传统上
- 直接用到英文 PPT 或商务跨文化场景，会被误读为"未完成"或"信息缺失"

### 5.6 与吕敬人 / 杉浦康平的根本分野
- **杉浦 / 吕敬人**：高密度、多层信息、东方丰盈
- **朱赢椿**：极简、留白、东方"虚"

**下游 skill 调用「中国风」时必须问清船长想要的是密度的东方还是留白的东方** —— 这是两个相反方向。

## 蒸馏到 PPT skill 的核心准则

1. **留白率 ≥ 60-80%**
2. **底色用宣纸米黄** `#F5F0E6`，**禁纯白**
3. **字体只用宋体**，**禁黑体作正文**
4. **字号梯度 2-3 级**，差距夸张（8pt vs 36pt 这种）
5. **微妙不对齐**（标题偏左、正文下沉、旋转 -1° 到 -3°）
6. **配色 ≤ 3 色**：墨黑 + 宣黄 + 朱砂（≤ 3% 占比）
7. **禁居中安全**、**禁炫技字体**、**禁渐变阴影 3D**
8. **适用边界**：单页深度阅读 / 封面 / 章节过渡 / 引用页；**绝不**用于演讲投影 / 数据 / 营销 / 多页连贯叙事
