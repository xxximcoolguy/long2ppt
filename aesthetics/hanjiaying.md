## 1. 视觉哲学

### MM-1 · 汉字即图形
拒绝把汉字当文字读。《天涯》15 年的核心实验：**让汉字在"读不出来"的状态下也能传递美感**。汉字的笔画、结构、肌理被当作纯粹视觉对象处理，**象形性与写意性优先于可读性**。

与靳埭强的分野：靳重"水墨意境"，韩**重"字本身"**。

### MM-2 · 道法自然 —— 东方哲思而非东方符号
援引《道德经》"道法自然"，但**反对将哲思翻译成龙凤祥云印章这种贴图式国风**。
**东方性是结构性的**（留白、层叠、虚实），**不是装饰性的**。

### MM-3 · 东西方对话而非选边
拒绝民族主义也拒绝全盘西化。**翻译者位置** —— 用国际化现代设计语言（Swiss + 模块化）作骨架，承载东方内容。

与同代人参照：
- Vignelli = 纯粹瑞士现代主义
- 原研哉 = 日本本土哲学当代化
- **韩家英 = 用国际语言承载东方内容**

### MM-4 · 形式实验主义
学界（祝帅）评价：贡献集中在"形式"。**用形式说话**，不是观念艺术家，不是商业品牌策略家。

## 2. 版式操作系统

| 手法 | 具体操作 |
|---|---|
| **汉字独占 50-80%** | 单字或词组被极度放大、剪切、解构、留白处理。**让网格服从字**（vs Swiss 让字服从网格）|
| **多重叠加（Overlay）** | 摄影 + 中文书法 + 印刷字 + 笔触四层叠合。**最多 4 层**，比杉浦克制 |
| **朱砂方印作锚点** | 朱红方印是与"中国身份"的连接器，**克制到 1 枚** —— 右下 / 左上居多，"压舱石"作用 |
| **网格 + 故意越界 5-15%** | 隐性瑞士网格 + 核心字打破网格、出血、跨页 —— **理性骨架 + 感性突破** |
| **大留白 + 单点强对比** | 背景常为米黄、宣灰、纯黑、纯白；主字与底强对比但不刺眼 |
| **图文比例 15-30 倍** | 中文主字号是辅助信息的 **15-30 倍** —— 绝对视觉层级 |

## 3. 颜色与字体

### 韩家英视觉签名三色

| 角色 | HEX | 用途 |
|---|---|---|
| **朱砂红** | `#B22222` / `#A52A2A` | 印章 / 强调字 / 点睛色块 |
| **墨黑**（偏暖，非纯黑）| `#1a1410` | 主体字 / 书法笔触 |
| **宣黄**（背景主色）| `#f0e6d2` / `#ebe0c8` | 大面积背景 |
| 辅助 | 灰阶 | 次级信息层 |
| 偶用 | 低饱和青、靛、赭 | 季节性变体 |

> "朱砂 + 墨 + 宣黄" 是他的视觉签名三色，与原研哉"全白系" / Vignelli"黑白红黄" 形成对照。

### 字体（无官方规范，基于作品归纳）

| 场景 | 中文 | 西文 |
|---|---|---|
| 主标题 | 宋体（方正书宋 / 华文宋）/ 定制书法 | Helvetica / Univers |
| 副标题 | 思源黑体 / 楷体 | Helvetica |
| 正文 | 宋体 | Helvetica |
| 装饰 | 篆书 / 隶书（< 5%）| 罕用衬线 |

**字体哲学：宋体承载"传统优雅"，黑体承载"当代力量"，二者并用而非二选一**。西文极度克制 —— **不和西文较劲，把创造力留给中文**。

## 4. 视觉禁忌

1. ❌ **反"假国风贴图"** —— 龙 / 凤 / 祥云 / 京剧脸谱 / 剪纸 / 青花 / 回字纹"中国符号大礼包"
2. ❌ **反"Helvetica 化的中文"** —— 把中文当中性字体用 = 廉价的国际化。中文有自己的呼吸节奏
3. ❌ **反过度装饰 / 过度信息** —— 与杉浦/吕的"信息丰盈"路线相反
4. ❌ **反"全盘西化"也反"民族主义"** —— 双重禁忌
5. ❌ **反印章滥用** —— 一枚是锚点，多枚是廉价的"中国味"
6. ❌ **反阴影、渐变、3D、neumorphism、过度圆角**
7. ❌ **反"设计稿即终稿"** —— 反复实验，《天涯》15 年就是一个长实验

---

## 5. 局限

1. **主要为印刷海报美学** —— 大部分代表作是 A1/B1 印刷海报。对屏幕动效、滚动叙事支持有限。**直接套到 PPT 时单页可用，连贯叙事易割裂**
2. **不适合数据可视化** —— 没有图表 / 表格 / 信息图范式，让位给 Tufte
3. **不适合营销爆款 / 小红书 / 抖音封面** —— 高级感来自克制，但算法推荐场景无点击率
4. **不适合企业 B 端模板** —— 不适合密集信息财报、年报、产品 spec。让位给 Vignelli
5. **不适合中文密度过高的长文** —— 版式预设"一字千金"，每页 20 字以内最佳；超过 200 字时画面平衡破坏
6. **依赖印刷工艺** —— 朱砂、宣纸肌理、烫印在屏幕上丢失约 30% 质感
7. **形式 > 观念** —— 学界批评："形式实验"高，观念性 / 社会批判性较弱

## 蒸馏到 PPT skill 的核心准则

1. **背景禁纯白**：用宣黄 `#f0e6d2` 或暖米
2. **主标题中文宋体**，独占 50-80% 视口
3. **一枚朱砂方印**（`#B22222`），右下或标题角，**绝不超过 1 枚**
4. **配色 ≤ 3**：朱砂 + 墨 + 宣黄
5. **西文只用 Helvetica 系**，体积极小作辅助
6. **网格 + 故意越界 5-15%**：理性骨架 + 感性突破
7. **主字 vs 辅信息字号 = 15-30 倍**
8. **适用边界**：单页海报 / 章节扉 / 引用页；**绝不**用于数据 / 营销 / B 端模板 / 长文密集
