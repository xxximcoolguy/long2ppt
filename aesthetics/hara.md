# 完整调研报告（按 5 层框架）

## 1. 视觉哲学（Mental Models）

### MM-1 · Emptiness / 空（うつ）—— 最核心的认知框架
【一手 · *Designing Design*, 2007】

Hara 区分两个概念：**Simplicity（简洁）** 和 **Emptiness（空）**。
- **Simplicity** 是 19 世纪西方现代主义的产物 —— 去除多余，追求理性效率。
- **Emptiness** 是日本传统的"空的容器"—— 不是没有，而是**可以容纳一切想象的空间**。

> "An empty vessel can hold anything. The image of a thing is more powerful than the thing itself." —— *Designing Design*

**操作含义：** 留白不是"省略"，而是**主动制造想象空间**。一张 PPT 上 80% 是白，那 80% 是设计的主体，不是背景。

### MM-2 · White / 白 —— 色彩的起点而非终点
【一手 · *White*, 2009】
- 白不是"无色"，而是**所有感知的母体（matrix of sensations）**
- 白是"还没有被定义的状态"，给观者留下完成作品的权利
- 一张白纸 ≠ 空白；它是充满潜能的场域

### MM-3 · Re-design —— 重新审视日常
【一手 · *Re-design* 2000 展览】
不要从零开始炫技，而是问"这个东西的本质是什么？能不能用更克制的方式呈现？"

### MM-4 · Senseware —— 媒介即讯息
【一手 · Tokyo Fiber 2007】
在 PPT 中，纸张质感、字体本身的呼吸感、留白的密度都是讯息，不只是文字内容。

### MM-5 · "This will do" 哲学（MUJI 内核）
不是"这个最好"（That's the best），而是"这样足够了"（This is enough）。反品牌符号化、反消费主义炫耀。

## 2. 版式操作系统（Heuristics）

### H-1 · 极致负空间 70-90% 留白率
【一手 · MUJI Horizon 系列】

经典 MUJI 海报：玻利维亚乌尤尼盐湖 / 蒙古草原。
- 画面中地平线 + 一个小人 = 占画面不到 10%
- 其余 90% 是天空 + 地面（接近纯色）
- logo "MUJI" 极小，放在角落

**CSS 落地：**
```css
.hara-frame {
  padding: 12vh 14vw;
  background: #fafaf7;  /* 暖白，非纯白 */
}
.hara-content {
  max-width: 42%;  /* 内容只占视觉宽度一半左右 */
}
```

### H-2 · 单点焦点构图（One Point in a Vast Field）
**一个小元素 + 巨大留白**，元素越小，留白的"重力"越大。
- 反例：把元素放大填满版面
- 正例：标题字号 28-40pt，周围留至少 3-4 倍空白

### H-3 · 不居中：偏左 / 偏下
- 主元素放在**视觉黄金分割点偏下**（55-62% 高度位置）
- 文字左对齐，左缩进约 8-15% 视宽
- 标题与正文之间**异常宽**的间距（标题字号的 3-5 倍）

```css
.hara-title {
  position: absolute;
  left: 12%;
  top: 58%;  /* 偏下而非居中 */
  font-size: 2.4rem;
  letter-spacing: 0.08em;
  margin-bottom: 4em;
}
```

### H-4 · 字号克制：拒绝"大字震撼"
【一手 · *Designing Design* 第 4 章】
- 标题与正文字号比一般 **1.4x - 1.8x**，几乎从不超过 2x
- 在 PPT 场景：标题 32-40pt，正文 18-22pt
- **绝不出现 80pt 巨字**

### H-5 · 行距宽松 —— 呼吸感
- 中文场景：line-height 1.8-2.2
- 段落间距 = 2-3 倍行高
- 字距：中文 0.05-0.1em，西文 0.02-0.05em

### H-6 · "信息少 = 重量大"
一页只放一个核心信息。信息密度低 ≠ 内容少，而是把内容**分散到更多页**。
**PPT 操作：** 30 页轻信息密度 > 10 页塞满文字。

## 3. 颜色与字体启发式

### 配色（**暖白绝不纯白 / 墨黑绝不纯黑**）

| 角色 | 色值范围 | 用途 |
|------|---------|------|
| 主白 | `#fafaf7` - `#f5f3ee` | 大面积背景，**绝不用 #ffffff** |
| 浅灰 | `#e8e6e0` - `#d4d2cc` | 分隔、次级元素 |
| 中性灰 | `#8a8782` | 正文 |
| 墨黑 | `#1a1a1a` - `#2b2926` | 标题、强调，**绝非 #000** |
| 唯一点缀 | 极弱浅色（淡米、灰蓝、淡褐） | 整页最多出现一次 |

**关键禁忌：** 不用 `#ffffff` 和 `#000000` —— Hara 的"白"和"黑"都是有温度的。

### "白是容器" 准则
- 同一页内**白的层次至少 2-3 个**（`#fafaf7` / `#f5f3ee` / `#ecebe6`）
- 用极微弱色差制造"近乎察觉不到"的分区，而非用线/框

### 字体系统

**中文场景：**
- 标题：思源宋体 / 方正悠宋（明朝体古典衬线）
- 正文：思源黑体 **Light / Regular**（绝不 Bold）
- **绝不混搭三种以上字体**

**西文：**
- 衬线：Bodoni / Didot / Garamond
- 无衬线：Helvetica Neue Light / Univers Light（Hara 本人 *Designing Design* 用 Univers）

**字重选择：** 大量使用 Light / Regular，**几乎不使用 Bold**。强调靠间距和位置，不靠字重。

### 字号阶梯
```
H1 标题：2.4rem (≈38pt)
H2 副题：1.6rem (≈25pt)
正文：  1.1rem (≈18pt)
注释：  0.85rem (≈14pt)
比例：1.0 / 1.3 / 1.6 / 2.4 —— 温和递进，无极端跳跃
```

## 4. 视觉禁忌（Anti-patterns）

【一手 · Hara 在 *Designing Design* 与访谈中明确反对】

1. ❌ **反视觉过载** —— 不在一页放多个焦点；不用渐变 / 阴影 / 玻璃质感 / 3D
2. ❌ **反"看得见的设计"** —— "设计应该消失。最好的设计是被感受到的，不是被看到的"
3. ❌ **反装饰性** —— 不用边框 / 分隔线（除非极细 0.5px 浅灰）/ 图标 / 背景图案
4. ❌ **反过度品牌化** —— MUJI logo 在海报中只占 1-2% 面积
5. ❌ **反纯黑纯白** —— `#000` 太冷漠，`#fff` 太医院化
6. ❌ **反超大标题字** —— 用留白制造重量，不用字号
7. ❌ **反 Bold** —— 几乎不用粗字重
8. ❌ **反三种以上字体混搭**

## 5. 局限性

| 不适用场景 | 原因 |
|----------|------|
| 数据密集型 / 财报 / 技术文档 | 留白 70% 意味着信息密度极低，30 页讲完别人 10 页的事 |
| 营销爆款 / 电商场景 | "这样就好"哲学与"立即购买"驱动冲突 |
| 信息海报 / 公共标识 | 需要高信息密度 |
| 数字投影 | sRGB 校色可能让 `#fafaf7` 暖白显示为脏白 |
| 高管简报 | "一页一信息"会让听众焦虑 |

**对中文场景特别说明：**
- 中文字符笔画密度天然高于日文假名和西文
- 用 Hara 比例直接套中文 —— **留白要再放大 10-15%**
- 中文宋体在小字号下笔画细节丢失，正文建议用思源黑体 Light

## 操作清单（给 PPT skill 用）

**必做：**
- 留白率 ≥ 70%
- 暖白背景（#fafaf7），非纯白
- 单页单焦点
- 字号克制（标题 ≤ 40pt）
- 行距 1.8-2.2，字距宽松
- 标题位置偏左偏下，非居中
- 字体：明朝/宋体 标题 + 黑体 Light 正文
- 一页最多 3 个色阶

**必不做：**
- ❌ 纯黑 #000 / 纯白 #fff
- ❌ Bold 粗字重
- ❌ 装饰元素（图标、边框、分隔线、背景图）
- ❌ 渐变、阴影、3D
- ❌ 多焦点、信息塞满
- ❌ 超大震撼标题
- ❌ 三种以上字体混搭
- ❌ 高饱和色（除非整套作品仅出现一次）

**适用判断：**
- 适合：品牌发布、文化项目、产品哲学阐述、读书分享、艺术展览、设计提案
- 不适合：数据报告、电商促销、技术 spec、信息海报、高密度商务汇报
