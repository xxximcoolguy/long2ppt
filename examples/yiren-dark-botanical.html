我的 AI 工具清单

MY AI STACK · LESS IS MORE

AI 工具是投资行为，不是消费行为。这是我这几年 ROI 最高的投资 —— 用几十到上千美金，拥有了全世界最好的几个"人"在帮你。

2025 上半年我搞了很多工具，现在反而越用越少 —— AI 到了做减法的阶段。

01 · TOOLS04 / 20
CORE STACK / 月支出不到 1000 美金

6 个核心 + Obsidian 工作台

01 · INTERACTION KING

Claude Max

跟"人"互动的王者
Claude Code 编程绝对最强；Cowork 做 PPT；财务分析、HTML、文案视觉化 —— 远远拉开差距。封号严重，又爱又恨。
02 · STRATEGY

ChatGPT Pro

Gemini + Notebook LM

学习神器（侧边栏 + Antigravity）
Google AI Pro 20 刀。AI 亦仁的标杆就是 Notebook LM —— 任何问题它都能给惊艳的回答。
04 · MOBILE

Manus

手机场景的"有经验员工"
不像 OpenClaw 要自己训练，Manus 是调教好的员工，布置任务一次出 PDF。AI 婚纱照调研就是它做的。
05 · INTEL

X + Grok

AI 经验的最大知识库
X 是 AI 实践最丰富的知识库；Grok 直接调取 X 所有数据 —— 等于我的"AI 经验查询器"。
06 · TRANSLATE

沉浸式翻译

全球信息无障碍
1000 块一年，涨到 1 万也得买。让我没有任何阻碍地浏览和学习全球信息。

决策者的四步框架

A DECISION-MAKER'S FRAMEWORK

市面上分享 AI 的主要是技术开发者和内容生产者。但我不是 —— 我是一个决策者。

公司老板、操盘手、一人公司，每天面对大量决策 —— 我们这种人怎么用 AI？这就是今天分享的视角。

02 · FLYWHEEL07 / 20
THE FOUR-STEP LOOP

信息 → 决策 → 执行 → 沉淀 = 飞轮

任何事我都按这个流程走 —— 一旦闭环，就形成飞轮。

01

信息源

给自己吃好东西 —— 国内、海外、专属数据三层。天天吃垃圾内容，思考质量一定不高。
02

思考决策

Grok / ChatGPT Pro / Agent / Notebook LM —— 复杂问题深度推理，把信息变成判断。
03

执行

能让 AI 干的我不干 —— Claude Code + Codex 双核，是 AGI 级别的通用 Agent。
04

沉淀复用

Obsidian + iCloud 本地化。数据归属于你，不归属于任何 AI 公司。
▽ ▽ ▽ 闭环之后，才是飞轮 ▽ ▽ ▽
02-01 · INPUT08 / 20
STEP 01 · YOUR DIET

信息就像你吃的东西

国内是二手信息居多；作为有追求的获取者，应该去看 X、看 YouTube。

国内

  • 生财有术：超级标 + 精华帖 + AI 亦仁问答
  • 微信群聊：导出 → PDF → Notebook LM 做深度分析（曾让 Notebook 出的课程框架秒杀团队）
  • 公众号：导出 PDF 灌 Notebook，研究透一个人 → 思维方式变成你的
  • 视频 / 电子书：Get 笔记 + 百度网盘 AI 笔记

海外 + 专属数据

  • X：沉浸式翻译 / 自己 API / Grok 调研
  • YouTube：Lex Fridman 等顶级访谈逐字稿全部能导入 Notebook
  • 海外社区：BlackHatWorld / IndieHackers / HN / Reddit
  • 最独家 → 专属数据：微信/飞书聊天 + 推特点赞 + Chrome 浏览 + AI 问答 全沉淀进 ObsidianAI 亦仁回答好，核心就是 9 年 150 万字的专属知识库。

02-03 · EXECUTE10 / 20
STEP 03 · LET AI DO IT

能让 AI 干的，我不干

核心工具就两个：Claude Code + Codex —— 它们已经是 AGI 级别的通用 Agent，不只是编程。

CASE 01 · 几百张发票

邮箱里几百封带发票邮件，Codex 一键搞定

"帮我下载邮箱里这几百封发票" → Codex 吭哧吭哧下载 + OCR 识别 + 整理成表格。以前极其耗时，现在一键交给财务。

CASE 02 · 微信 H5 抓包

一百多份医院诊断数据，只在微信 H5 里

没有网页版的微信客户端数据本来很难拿。Codex 直接帮我抓包 —— 我不知道它怎么实现的，反正全下来了。

Claude Opus 4.6 GPT-5.4 Opus 4.7 本周出 → 学 AI 编程不需要代码基础，会有学习成本，但比你想象的低

FINALE · MINDSETS18 / 20
FINAL · FOUR PRINCIPLES

给所有人的四个心法

PRINCIPLE 01 · FOCUS

聚焦

不需要抓很多，一个深耕下去就够了。一个项目跑通闭环 (几百几千)，绝对不要轻易换 —— 任何一个项目都有机会赚到百万级以上。

PRINCIPLE 02 · OUTSIDER

边缘人理论

没项目、不懂 AI、没赚很多钱 —— 不要焦虑，这反而是优势。有项目的人没时间学 AI；每次大机会来的时候，往往是边缘人先进场。

PRINCIPLE 03 · ACT

先干再说

"先上课、先学习、有体系才开始" —— 这是做题家思维。先躬身入局，摸到枪。Connect the Dots —— 你做的每件事最后都会在意想不到的点上呈现。

PRINCIPLE 04 · CIRCLE

换圈就是改命

命运改变 ← 行动改变 ← 认知改变 ← 视野改变。99.99% 的人都是看见才相信。每个圈子背后都是一群思维方式的合集 —— 投资自己的脑子，是最重要的投资。
